# 策略回测可视化游戏化平台

一个基于 Vue 3 + TypeScript 的策略回测前端平台，支持多种策略角色，提供实时回测模拟和数据可视化。

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- Pinia
- Element Plus
- ECharts + vue-echarts
- Vue Router

## 页面说明

### 1. 策略配置页面 (`/config`)
- 展示 4 种策略角色卡牌
- 趋势追踪者：跟随市场趋势
- 均值回归者：价格回归均值时交易
- 突破猎手：捕捉突破行情
- 防守型资金管理者：稳健保守
- 支持选择策略并跳转回测

### 2. 历史行情页面 (`/market`)
- K 线图展示历史价格走势
- MA5、MA20 均线指标
- 成交量柱状图
- 行情数据表格展示

### 3. 回测执行页面 (`/backtest`)
- 回测控制：开始/暂停/重置/快进
- 速度调节
- 实时指标统计：总资产、收益率、最大回撤、胜率等
- 收益曲线图（策略 vs 基准）
- 交易明细表格
- 持仓详情展示

### 4. 策略排行榜页面 (`/ranking`)
- Top 3 策略展示
- 完整排行榜表格
- 策略选择建议

## 核心回测算法

### 回测思想
按时间顺序遍历历史行情数据，在每个时间点：
1. 根据策略规则生成交易信号（买入/卖出/持有）
2. 执行交易，更新资金和持仓
3. 计算手续费
4. 记录组合状态
5. 更新收益率、最大回撤、胜率等指标

### 策略信号生成

#### 1. 趋势追踪策略
- 价格上穿 MA20 且 MA10 > MA20 时买入
- 价格下穿 MA20 时卖出

#### 2. 均值回归策略
- RSI < 30 或价格跌破布林带下轨时买入
- RSI > 70 或价格突破布林带上轨时卖出

#### 3. 突破猎手策略
- 价格突破前 20 日高点时买入
- 价格跌破前 20 日低点时卖出

#### 4. 防守型策略
- 价格上穿 MA30 且 MA10 > MA30 时买入
- 价格跌破 MA30 的 95% 时卖出
- 单笔下注不超过总资金 30%

### 指标计算

#### 收益率
```
收益率 = (当前总资产 - 初始资金) / 初始资金 * 100%
```

#### 最大回撤
```
最大回撤 = max( (历史高点 - 当前值) / 历史高点 * 100% )
```

#### 胜率
```
胜率 = 盈利交易次数 / 总交易次数 * 100%
```

## 运行步骤

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:5173` 启动（或 Vite 分配的其他端口）。

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 在策略配置页面选择一个策略角色
2. 可以先去历史行情页面查看价格走势
3. 进入回测执行页面，点击「开始」按钮启动回测
4. 观察实时指标变化和收益曲线
5. 可以调整回测速度或使用快进功能
6. 查看交易明细了解详细交易记录
7. 在策略排行榜页面对比不同策略表现
